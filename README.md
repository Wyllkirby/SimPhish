## 📝 项目简介

SimPhish 是一个专为企业内部安全意识培训设计的钓鱼邮件模拟平台。通过模拟真实的钓鱼攻击场景，帮助组织提高员工的安全意识，识别和防范网络钓鱼威胁。

### 核心价值

- **教育导向**：提升员工安全意识
- **安全可控**：确保测试环境安全
- **数据追踪**：记录关键行为数据
- **灵活定制**：自定义邮件模板和钓鱼页面

---

## ⚙️ 功能特性

### 📧 邮件管理

- **模板编辑**：支持 HTML 和富文本编辑器
- **变量替换**：内置多种变量（邮箱、姓名等）
- **附件支持**：可添加最多 10 个附件，最大 10MB
- **SMTP 配置**：支持多种加密方式（SSL/TLS、STARTTLS）
- **发送控制**：支持并发控制和速率限制

### 🌍 钓鱼页面

- **页面编辑**：支持自定义 HTML 内容
- **变量支持**：页面内容支持变量替换
- **服务器配置**：可配置独立的钓鱼服务器地址
- **重定向**：提交后可重定向到指定 URL

## ⚙️ 配置说明

安装完成后，您需要配置一些基本设置，以确保应用正常运行。

1. **SMTP 设置**：在设置页面中，输入您的邮箱服务器信息。
2. **钓鱼页面设置**：自定义您的钓鱼页面内容。
3. **用户管理**：添加您的团队成员以便进行测试。

详细的设置步骤请查看 [使用指南](#使用指南)。

---

## 📚 使用指南

1. **创建钓鱼邮件**：选择邮件模板并添加接收人，设置发送时间。
2. **发起钓鱼攻击**：点击发送按钮开始模拟攻击。
3. **查看结果**：通过分析仪表板查看员工的行为数据。

## 📜 API 文档

SimPhish 还提供了 API 接口，支持二次开发。您可以通过 API 获取结果数据，创建自定义功能。例如，可以使用 GET 请求获取用户数据。

## 📊 数据管理

SimPhish 会详细记录每次钓鱼的结果。您可以查看员工打开邮件、点击链接和提交表单的行为数据。数据以图表形式展示，易于理解。
